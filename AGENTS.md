# polotno — Документация агента и статус проекта

**polotno** — веб-приложение для рисования и заметок с бесконечным холстом в стиле Apple HIG (аналог Excalidraw / tldraw).

---

## 🛠 Технологический стек
- **Фреймворк**: Next.js 14+ (App Router, TypeScript)
- **База данных**: MongoDB Atlas + Mongoose
- **Стилизация & Шрифты**: Tailwind CSS v4, Google Fonts (`Mulish`, `Geist`)
- **Анимации**: Framer Motion (`src/lib/motion.ts` — spring-пресеты Apple HIG 150–250 мс)
- **Рендер холста**: HTML5 Canvas + `roughjs` (ручной эскизный стиль) + `perfect-freehand` (плавные линии карандаша)
- **Состояние**: Zustand (`useCanvasStore.ts`)
- **Иконки**: `lucide-react`
- **Репозиторий**: https://github.com/Kingrane/polotno

## ✅ Что уже сделано (Этап 1, 2 & 3: Гостевой MVP + Облачные доски & Авторизация)

### 🎨 1. Движок холста (`src/engine/`)
- **Бесконечный холст**: Панорамирование (пробел + драг, средняя кнопка мыши, тачпады) и перехваченный зум (`Ctrl + колесико`, кнопки в шапке, горячие клавиши `Ctrl +`, `Ctrl -`, `Ctrl 0`).
- **Мобильный мультитач**: Нативная поддержка пальцевого масштабирования (pinch-to-zoom) и панорамирования двумя пальцами (`touchstart`, `touchmove`, `touchend`).
- **Инструменты рисования**:
  - `Выделение (1 / V)` — передвижение, масштаб за 8 точек (с сохранением пропорций для картинок/Shift), поворот, множественный выбор рамкой (Marquee).
  - `Панорамирование (2 / H)`
  - `Карандаш (3 / P)` — сглаженные stroke-линии через `perfect-freehand`.
  - `Фигуры`: Прямоугольник (`4 / R`), Овал (`5 / O`), Ромб (`6 / D`), Линия (`7 / L`), Стрелка (`8 / A`).
  - `Текст (9 / T)` — быстрая вставка текста одиночным кликом на холсте.
  - `Ластик (0 / E)` — быстрое стирание элементов.
  - `Картинка` — загрузка из файла, свободное перемещение и масштабирование.

### ☁️ 2. База данных & Динамические облачные доски (`/board/[id]`)
- **MongoDB Atlas Integration**: Подключение через Mongoose (`src/lib/db.ts`) с обработкой сокетных соединений в serverless-среде Next.js.
- **Динамический роутинг `/board/[id]`**: Создание уникальных коротких ссылок для каждой доски.
- **Автоматическая синхронизация**: Автосохранение изменений в MongoDB с дебаунсом 800 мс и живым статусом `Облако / Сохранение...`.
- **Плавающий виджет облака (`CloudBoardWidget.tsx`)**: Находится в правом нижнем углу экрана (`bottom-4 right-4`), появляется при включении открытого режима в модалке «Поделиться». Позволяет переименовывать доску и создавать новые облачные доски в 1 клик.

### 🔑 3. Авторизация пользователей (`AuthModal.tsx`)
- Маршруты `/api/auth/register` и `/api/auth/login`.
- Окно регистрации и входа с валидацией данных.
- Отражение имени вошедшего пользователя в шапке сайта.

### 🌟 4. Секретный код `arbuz` & Pro-функционал
- Ввод промокода `arbuz` в `ProModal.tsx` активирует Pro-статус (`isPro: true`).
- Открытие пипетки «Свой цвет» (`<input type="color">`) в `StylePanel.tsx`.
- Золотой индикатор `PRO активен` в шапке.
